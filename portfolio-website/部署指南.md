# 网站部署指南

## 部署前准备

### 1. 准备个人照片

在部署网站之前，你需要准备以下6张照片：

| 文件名 | 用途 | 要求 |
|--------|------|------|
| `profile-photo.jpg` | 个人资料页头像 | 脸部清晰可见，正方形最佳 |
| `photo1.jpg` | 画廊-个人头像 | 脸部清晰可见 |
| `photo2.jpg` | 画廊-上海之旅 | 旅行照片 |
| `photo3.jpg` | 画廊-摄影时光 | 摄影相关 |
| `photo4.jpg` | 画廊-成都记忆 | 旅行照片 |
| `photo5.jpg` | 画廊-阅读时光 | 阅读/学习相关 |

**照片要求**：
- 格式：JPG
- 大小：建议 800x600 像素或更大
- 必须是你自己的原创照片
- 脸部在头像照片中必须清晰可见

将照片放入 `images` 文件夹中。

### 2. 自定义内容（可选）

你可以根据个人情况修改以下内容：

- **姓名**: 在 `profile.html` 中搜索"张明"并替换为你的真实姓名
- **联系方式**: 在 `contact.html` 中更新邮箱、电话等信息
- **个人介绍**: 修改各页面的文字内容
- **照片说明**: 在 `gallery.html` 中修改照片的描述文字

## 部署到 InfinityFree

### 步骤 1: 注册账号

1. 打开浏览器，访问 https://www.infinityfree.net/
2. 点击右上角的 "**Sign Up**" 按钮
3. 填写注册信息：
   - 用户名（Username）
   - 邮箱地址（Email）
   - 密码（Password）
4. 点击 "**Create Account**"
5. 查收验证邮件，点击链接验证邮箱

### 步骤 2: 创建网站

1. 登录 InfinityFree 控制面板
2. 点击 "**New Account**" 或 "创建新账户"
3. 选择域名类型：
   - **选项A**: 使用免费子域名（推荐）
     - 选择 "Subdomain"
     - 输入你想要的子域名前缀（如：zhangming-portfolio）
     - 选择后缀（如：.epizy.com 或 .rf.gd）
   - **选项B**: 使用自己的域名（如果有）
4. 等待账户创建完成（约1-2分钟）

### 步骤 3: 上传文件

#### 方法一：使用在线文件管理器（推荐）

1. 在控制面板中，点击 "**Online File Manager**"
2. 找到并进入 `htdocs` 文件夹（这是网站的根目录）
3. 删除默认的 `index.html` 文件
4. 上传你的文件：
   - 点击 "**Upload**" 按钮
   - 选择所有网站文件（保持文件夹结构）
   - 确保 `css`、`js`、`images` 文件夹都上传

#### 方法二：使用 FTP

1. 在控制面板中找到 FTP 账户信息
2. 使用 FTP 客户端（如 FileZilla）连接
3. 连接到服务器后，进入 `htdocs` 文件夹
4. 上传所有文件

### 步骤 4: 验证部署

1. 在浏览器中访问你的网站 URL
   - 例如：`http://zhangming-portfolio.epizy.com`
2. 检查以下内容：
   - 所有4个页面都能正常访问
   - 导航栏能正常跳转
   - 照片能正常显示
   - JavaScript 功能正常工作

## 常见问题

### Q1: 照片显示不出来怎么办？

**解决方法**：
1. 检查照片文件名是否正确（区分大小写）
2. 确认照片已上传到 `images` 文件夹
3. 检查照片格式是否为 JPG
4. 在浏览器中按 F12 打开开发者工具，查看控制台错误信息

### Q2: CSS样式没有生效？

**解决方法**：
1. 确认 `css/style.css` 文件已上传
2. 检查HTML文件中CSS链接路径是否正确
3. 清除浏览器缓存（Ctrl+F5）

### Q3: JavaScript功能不工作？

**解决方法**：
1. 确认 `js/main.js` 文件已上传
2. 按 F12 打开浏览器控制台，查看是否有错误信息
3. 检查浏览器是否禁用了JavaScript

### Q4: 如何更新网站内容？

**解决方法**：
1. 修改本地文件
2. 重新上传到 `htdocs` 文件夹，覆盖旧文件
3. 刷新浏览器查看更改（Ctrl+F5）
